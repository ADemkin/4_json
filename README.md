# Prettify JSON

Prints a json file in a human-friendly and easy readable format (pretty print).

Anton Demkin, 2017
antondemkin@yandex.ru

# Quickstart


usage:
```
python pprint_json.py [file.json]
```
example:
```
python pprint_json.py example.json
```
